运行faiss的时候遇到Cannot load libmkl_avx2.so or libmkl_def.so.问题

  • 无需重新安装, 添加命令 到~/.bashrc中
  • 问题: 是因为环境配置过程中mkl库与其他库的依赖关系出现混乱导致的, 即mkl动态库对应有多处路径
  • LD_PRELOAD是个环境变量, 用于动态库的加载, 动态库加载的优先级最高, 一般情况下, 其加载顺序为 因此使用最高级别的直接指定, 由于是anaconda中的python内部报错, 所以指定anaconda/lib
